== Installation ==
1. Primary: == Installation ==
	- In your admin panel, go to Appearance -> Themes and click the 'Add New' button.
	- Type in Webart in the search form and press the 'Enter' key on your keyboard.
	- Click on the 'Activate' button to use your new theme right away.
	- Go to https://themefreesia.com/theme-instruction/webart/ for a guide on how to customize this theme.
	- Navigate to Appearance > Customize in your admin panel and customize to taste.

2. Alternate: Unzip the template file (webart.zip) that you have downloaded. Via FTP, upload the whole folder (webart) to your server and place it inside the http://yourdomain.com/wp-content/themes/folder(webart). Do not change directory name. The template files should thus be here now: http://yourdomain.com/wp-content/themes/webart/index.php (for example).
	- Log into your WP admin panel and click on the Appearance -> Theme tab. Now click on the Webart theme to activate it.
	- Want to enjoy Webart feature. Go to  Dashboard -> Appearance -> Customize and play with Webart Options
	- The theme is offered free of charge. If someone asked money for it, someone just tricked you.

	Note: Parent Theme Photograph must be installed to work with Webart Child theme
